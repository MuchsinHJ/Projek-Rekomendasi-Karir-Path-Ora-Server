# Projek-Rekomendasi-Karir-Path-Ora

mengembangkan Platform End-to-End Kesiapan Kerja yang dilengkapi dengan dashboard strategis dan fitur rekomendasi jalur karir otomatis. Platform ini akan memanfaatkan data CV (dataset Kaggle) untuk menganalisis kompetensi pengguna menggunakan pendekatan data science dan machine learning. Hasil analisis akan ditampilkan dalam bentuk visualisasi yang informatif, serta memberikan rekomendasi karir yang relevan dan personal.
